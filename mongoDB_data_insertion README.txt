Manually inserting data in Mongo Compass HOWTO

1. Open Mongo Compass
2. Connect to a server or cluster using the new connection field
3. Once connected, create or open an existing database
4. In the database, create a collection
5. After the collection is created, click the ADD DATA button
6. There are two options for manually uploading data: importing a file (.csv,.json) and inserting a document.
   Choose the option most fitting to the given data
